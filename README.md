#### Technologies Used
ReactJS: For building the user interface.
CSS3: For styling the page.
HTML5: For the structure of the webpage.

#### Folder Structure

- public/: Contains static files like the index.html and assets (images, icons).
- src/: Contains all the React components and styles.
- components/: Contains individual React components like the Hero Banner.
- App.js: Main entry point for the React app.
- index.css: Global CSS styles.
- assets/: Image files and other static assets used in the app.


#### Customization

- You can easily modify the hero banner's content:
- Text: Change the main heading (<h1>) and subheading (<p>) text in the HeroBanner.js component.
- Image: Replace the background image by updating the hero-background.jpg image file in the assets/ folder.
- Button: Update the button text and link by modifying the button component in the Hero Banner.
